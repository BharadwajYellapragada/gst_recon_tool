GST Reconciliation Tool

Reconcile GSTR-2A against your Purchase Register, track late filers, and export ready-to-use Excel reports — for Windows.

Download for Windows
Windows 10/11 · no Python required · ~33 MB installer

After installing

  1. Run the installer (Windows may show an "unknown publisher" warning — click "More info" → "Run anyway").
  2. On first launch, the app shows a Machine ID — send it to whoever gave you this software.
  3. They'll send back an Activation Key. Paste it in to unlock the app on your computer.
  4. Set a PIN on first run — required every time you open the app.
